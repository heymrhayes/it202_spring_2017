Activity: jQuery Basics

In this activity, we'll explore some basics of jQuery by typing commands in the console and then by creating a simple page that includes jQuery commands.

Instructions

Part 1 - Commands in the console

  1. Open a new browser tab and navigate to http://www.uic.edu/
  2. Open the JavaScript console.
  3. Write and execute a jQuery selector to select all paragraph components on the page.
  4. Note the display in the console; you should see an array of elements (if your selector is correct.)
  5. Expand that array to see the individual elements, and note that as you mouseover each one, it will be highlighted in the browser window.
  6. Write a jQuery statement to hide all those paragraph elements.
  7. Write a jQuery statement to set the color of all h2 elements to green.
  8. Write a jQuery selector to select all elements with a class of home-list-header
  9. Write a jQuery statement to add a click event handler to those elements with an anonymous function to set their font-size to 100px. What if we only wanted to change the specific item we clicked on?

Part 2 - Add some jQuery to a web page

  1. In c9, create a new file using the html template, name it jquery_basics.html, and save it in a folder named activities (this should be a sibling to your labs folder.)
  2. Add a paragraph element with the content Hello jQuery; give it an id attribute with the value p1.
  3. Add a button element with the value Click me!; give it an id attribute with the value button1.
  4. At the end of the body element, add a script tag to include the jQuery library file; use a CDN URI. Test for errors.
  5. Below that, create another script tag, and create the main jQuery function for your page. Test.
  6. Create a mouseover event handler for the paragraph (use an id selector); add an anonymous function that sets the color to blue.
  7. Create a mouseout event handler for the paragraph (use an id selector); add an anonymous function that sets the color to black.
  8. Create a click event handler for the paragraph (use an id selector); add an anonymous function that increases the font size by 6px each time.
  9. Create a click event handler for the button (use an id selector); add an anonymous function that hides the paragraph element.
