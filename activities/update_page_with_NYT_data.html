Activity: Updating a page with NYT data

In this activity, we'll use data retrieved from the NYT Article Search API to update a page with the headline, byline, publication date, lead paragraph, image and link to the NYT site for each article returned.

Instructions

Part 1 - Examining the API

  1. Use the NYT API console to test your API key and to see the $.ajax script to use.
  2. Explore the data retrieved to see how to access the elements we need.

Part 2 - Adding script to a file

  1. Navigate to https://heymrhayes.github.io/it202_spring_2017/examples/it202_jumbotron.html, copy the source and use it to create a new file in your activities folder.
  2. Add a script element at the end of body element.
  3. Paste the $.ajax code from the NYT console into the block.
  4. Add a console.log statement as the first line of the done event handler to output the returned data.

Part 3 - Using $.each to loop through the data

  1. The jQuery $.each() utility method (http://api.jquery.com/jQuery.each/ lets us easily loop through objects and arrays. Add an $.each() statement to the done event handler to loop through the returned data.
  2. $.each() exposes two values for us to use inside the callback function - the index position and the value at that position. Modify the callback function with a console.log statement to output the pub_date value.

  3. The logic will be: find an element to use as a template, clone it, update the clone's elements with data values, and insert that clone into the DOM at the appropriate point.
