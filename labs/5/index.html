Lab 5: Firebase Simple Chat example

Firebase is a Google platform that speeds app development by providing some commonly needed server-side functionality (user authentication, data storage, file storage, even hosting) with just a few API calls.

In this lab, you'll work through their Simple Chat example to learn some of the basic features.

Instructions

Your chat app should be accessible at lab5.html.

  1. Navigate to the Simple Chat Code Lab at https://codelabs.developers.google.com/codelabs/firebase-web/#0.
  2. In step 5, if you're using c9, you'll run firebase login --no-localhost. The documentation assumes you're developing on your local machine, and the standard login command would launch a browser window and ask you to login. Running in "no-localhost" mode will provide a URL that you open separately.
  3. Ignore step 6: c9 is your server for now, so just preview your page as you've previously done.
