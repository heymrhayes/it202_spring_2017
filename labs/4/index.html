Lab 4: Canvas Manipulation

The HTML5 Canvas element lets us to do sophisticated drawing and animation directly in a standard HTML element, allowing us to do gaming and visual representations which are interactive and data generated.

In this lab, you'll create a page with multiple canvas elements which demonstrate a variety of drawing, animation and event handling features. You'll combine these features to make a game in a later project.

Instructions

You'll put all these elements in single file named lab4.html.

Part 1 - Simple text and shapes

  1. Create a canvas element with an id of canvas1, a width of 500 and a height of 300; make the background color the UIC dark blue on page 68 of the UIC Brand Binder
  2. Write JavaScript to add a 100 pixel radius circle in the center, with a fill color of UIC red from the binder.
  3. In the approximate center of the circle, write your initials and, below that, "UIC". Use the Helvetica font, make the text solid white, and choose a size so that the text fills about half the height of the circle.

Part 2 - Function make canvas work easier

  1. Create a function named draw, and move the code you wrote in step 1 into it.
  2. Create another function named clear that clears the canvas.
  3. Hook up keyboard event handlers such that the "X" keydown event clears the canvas and the "U" keydown event draws the circle and letters.

Part 3 - Animation

  1. Create another canvas element with the id canvas2, a width of 500, a height of 300 and any background color you want.
  2. Create a function named drawCircle that draws a 50 pixel radius circle on this canvas element.
  3. Add a requestAnimationFrame call at the end of the function (so that it continues), and add one outside the function to start the animation.
  4. Create a global variable circleX and initialize it to the center of this canvas. Change your drawCircle function so that it uses that variable to determine the center point. And add a line to the start of the function to increment the variable. Verify that the circle is moving.
  5. Update your function to handle "collision" with the vertical edges of the canvas. The circle should appear to bounce off the left and right edges.

Part 4 - A basic controller

  1. Create a global circleY variable, and update your function to use that value for the circle's position.
  2. Add event handlers to the up and down arrow keys to change the value of that varible. Verify that you can move the circle vertically.
