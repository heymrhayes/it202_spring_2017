Lab 6: Getting Started

Heroku is an application-hosting service that allows us to focus on software development and minimize our time on infrastructure maintenance. One of the key benefits is the ease of scaling.

Instructions

  1. Navigate to the tutorial at https://devcenter.heroku.com/articles/getting-started-with-nodejs#introduction.
  2. Work through the tutorial. The tutorial assumes you're working on a local machine, so ignore the open commands to launch your page; you'll run the application on c9 and preview the application.
  3. Create a file named lab6.html; it's content should be a link to your Heroku app.
  4. Push you lab6 up to GitHub.
