Lab 7: Intro to IndexedDB

"IndexedDB is a low-level API for client-side storage of significant amounts of structured data...." MDN IndexedDB

In this lab, you'll get a little practice in using the Dexie IndexedDB wrapper (so that we can avoid all the low-level details.)

Instructions

  1. Navigate to Dexie.js at http://dexie.org/.
  2. Click the Get Started in 30 Seconds button.
  3. Copy the content and save it as lab7.html.
  4. Run the page and see the alert; change that alert to a console.log, and verify that it's working.
  5. Open the developer console to the Application tab, and look for the IndexedDB data you created.
  6. Modify the code to add a different store with information you want to store.
  7. Modify the code to add records to that new store, and verify that it's visible in the developer console.
