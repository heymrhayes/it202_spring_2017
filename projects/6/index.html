Project 6: Intro to React

React is a JavaScript library for building user interfaces, and because it is so widely used, you want to be familiar with the basics.

You'll find some good tutorials on the basics created by Kirupa Chinnathambi and his team at https://www.kirupa.com/react/

You'll work through the set specified below, and you'll create the indicated file for each tutorial. (These can be basic HTML pages; no framework is required.) If a tutorial continues from the prior one, just copy over the file contents as a starting point for the next tutorial.

On the Building your First React App tutorial, two methods for working with JSX are mentioned. You'll be using the second method: Let your browser automatically convert JSX to JavaScript at runtime.
