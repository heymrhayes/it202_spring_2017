Project 2: Data App

In this project, you'll demonstrate your ability to accept user input, make AJAX calls and update the DOM with both text and a visualization.

You'll pull data on requests that are "open" from the City's "311 Service Requests - Pot Holes Reported" dataset at https://data.cityofchicago.org/Service-Requests/311-Service-Requests-Pot-Holes-Reported/7as2-ds3y.

Your page will include the following display elements:

  1. A table with the total number of potholes by community area,
  2. a piechart of that data (ordered by number of open requests), and
  3. a heatmap of the open pothole requests.

Your styling doesn't have to be fancy, but it should look like someone cared.

Instructions

  1. Get started early.
  2. Ask questions on Piazza (I don't expect you to have all the answers.)
  3. Test step by step.
  4. Get one of the outputs working before moving on to another.
  5. Ask a classmate or friend how your app can be better.
  6. Save and push to your GitHub repo.

Evaluation

The evaluation and scoring will be based on completeness and appearance; 25% for each of the display elements and for the overall look and feel.
